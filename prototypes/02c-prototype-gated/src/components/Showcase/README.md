# Showcase Page

Design system documentation page showcasing all UI components and design tokens.

## Usage

Import and render the `ShowcasePage` component:

```tsx
import { ShowcasePage } from '@/components/Showcase'

function App() {
  return <ShowcasePage />
}
```

## What's Included

### Design Tokens
- **Color Palette**: All color scales (Brown, Gray, Green, Orange, Yellow, Purple)
- **Semantic Colors**: Primary, Secondary, Success, Warning, Error, etc.
- **Typography**: All text styles (H1-H3, Body, Caption, Numeric)
- **Border Radius**: Card, Sheet, Pill, Chip, Input
- **Shadows**: Card, Sheet, Tile

### UI Components
- **Card**: Default, Interactive, Expandable variants
- **IconButton**: All variants (default, primary, danger, ghost) and sizes
- **ProgressRing**: With different progress states
- **MacroBar**: Horizontal progress bars with different colors
- **FoodTile**: Food selection tiles
- **MealCard**: Logged meal display cards
- **Toast**: Interactive toast notifications
- **BottomSheet**: Modal/drawer component

All components are interactive and demonstrate their various states and behaviors.
